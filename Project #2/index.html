About The Project
In this project i used HTML, CSS3, TypeScript ,JQueary, Bootstrap , AJax , Events and Rest API.
In Home page you can see about 100 Virtual Coins and see their values in USD, EUR and NIS. To see this click on More Info button.
You can add a button to a list that can conation up to 5 coins and see their values charts in live. To see it add the coin to the list by clicking the toggle button at the side of the coin name and switch to Live Report tab.
You can search for a coin by using the search option (even coins that you don't see in the home page).
